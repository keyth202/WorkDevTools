What is the difference between horizontal and vertical scaling?

Click to reveal answer

Answer:

Horizontal scaling (scaling out) adds more instances of the same size to handle increased load. Vertical scaling (scaling up) increases the capacity of an existing instance (e.g., more CPU, RAM).
